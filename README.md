# 🚀 SATSET (Sarana Aktivitas Terpadu & Sinkronisasi Ekosistem Terdata) - Frontend

**SATSET** Frontend adalah antarmuka modern, responsif, dan premium yang dirancang untuk memberikan pengalaman pengguna yang inklusif dan efisien. Dibangun dengan Vue 3 dan Tailwind CSS untuk menjamin kecepatan dan kemudahan navigasi.

> **Tagline**: *"Kolaborasi Cepat, Dokumentasi Tertata"*

---

### ✨ Keunggulan Frontend:
- **Vue 3 + Vite**: Startup server kilat dan hot-reload yang sangat responsif.
- **Tailwind CSS & Glassmorphism**: Desain antarmuka premium dengan nuansa transparan dan modern.
- **State Management (Pinia)**: Pengelolaan data aplikasi yang terstruktur dan reliable.
- **Responsive Design**: Optimal diakses dari berbagai perangkat (Desktop, Tablet, Mobile).
- **Interactive UI**: Micro-animations dan transisi halus untuk meningkatkan *user experience*.

## 🛠 Persiapan Lokal

### 1. Prasyarat
Pastikan kamu sudah menginstal:
- **Node.js** (versi 16+ / LTS)
- **NPM** atau **Yarn**

### 2. Instalasi
Clone repositori dan masuk ke direktori proyek:
```bash
git clone https://github.com/mhdaliffarhan/satset-frontend.git
cd satset-frontend
```

Instal dependensi:
```bash
npm install
```

### 3. Konfigurasi Environment (`.env`)
Buat file `.env` di root direktori:
```env
VITE_API_BASE_URL=https://api.satset.statsntb.id
```

### 4. Jalankan Aplikasi
Nyalakan server development:
```bash
npm run dev
```
Aplikasi dapat diakses di `http://localhost:5173`.

### 5. Build untuk Produksi
Jika ingin melakukan build statis:
```bash
npm run build
```
File hasil build akan berada di folder `/dist`.
